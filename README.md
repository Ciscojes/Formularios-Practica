# 📋 Formularios HTML5 e Inputs

Proyecto desarrollado como práctica de HTML5 para aprender la creación de formularios y el uso de los diferentes tipos de inputs disponibles en HTML5.

## 🚀 Características

- Formularios HTML5.
- Etiquetas `<form>` y `<label>`.
- Inputs de diferentes tipos.
- Validaciones nativas de HTML5.

---

## 🛠️ Tecnologías Utilizadas

- HTML5
- JavaScript

---

## 📚 Tipos de Inputs Practicados

- Text
- Email
- Checkbox
- Radio
- Button

## 🎯 Objetivos de Aprendizaje

- Comprender la estructura de los formularios HTML.
- Utilizar correctamente la etiqueta `<form>`.
- Asociar campos mediante `<label>`.
- Aplicar validaciones usando atributos HTML5.
- Conocer los distintos tipos de inputs y sus usos.
- Organizar proyectos utilizando herramientas modernas de desarrollo.

## 📸 Vista Previa

Este proyecto contiene ejemplos prácticos de formularios HTML5 y diversos tipos de campos de entrada para la captura de información de usuarios.
